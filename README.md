# Typescript config

> Shared Typescript config for all [@platadev](https://github.com/platadev) projects

## Installation

```bash
  yarn add @plata/tsconfig
```

then in `tsconfig.json`

```json
{
  "extends": "@plata/tsconfig/tsconfig.json",
  "compilerOptions": {
    "outDir": "dist"
  }
}
```
